Step 2 — set compartment_canonical on every biological_state.
  If the biological_state has a subcellular_location, set compartment_canonical to the matching
  canonical_name from the compartments[] list. If no canonical match exists, set it to "".

Step 3 — set subcellular_location on every biological_state.
  Even if the state is only implied (e.g., "cytosolic pyruvate" implies a cytosolic state),
  record the location verbatim in subcellular_location.
